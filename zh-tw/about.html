# 📋 背景

作為一位充滿熱忱的網頁開發者，我擁有穩固的HTML、CSS和JavaScript基礎，希望運用這些知識來創建視覺上吸引人且具有互動性的網站。我是一位快速學習者，熱衷於了解新興的網頁技術（例如React和Vue）。此外，我具備優秀的團隊合作能力，並致力於產出高質量的程式碼。

我的專業背景涵蓋網頁創建、UI開發和戰略行銷。在學期間在加拿大政府中兼職擔任網頁發佈員，確保網頁內容符合 Web Content Accessibility Guidelines (WCAG) 標準。我在使用 Adobe Experience Manager (AEM) 製作客製的網頁方面表現優越，總計更新和發佈1,000多頁網頁內容，其中包含有效解決無效連結並優化了內容的可訪問性。

在此之前，我在Agility PR Solutions實習中磨練了我的UI開發技能，運用Angular、TypeScript、HTML、SCSS 和 JavaScript 解決了UI的不一致性並無縫地整合了新設計的元件，例如按鈕、輸入框、下拉選單、側邊攔等等。我的經驗還涵蓋了行銷，我曾在星展銀行（台灣）策劃全面的行銷策略，善用總部資源，在台灣分支首度執行內容行銷企劃，共產出21篇金融教育文章和影片，持續取得令人印象深刻的成果。

最近，我積極地透過參與 Front-end Mentor Challenges來提升我的前端開發技能。我運用 HTML、SCSS、React、Vue 和 Vanilla JavaScript 創建了多個網頁應用程式和互動工具，包括一個小費計算器 和互動式價格元件。這些專案展現了我在根據提供的模型和規格開發使用者介面方面的熟練程度。

我能在協作項目中蓬勃發展，並在動態環境中表現卓越。如果您有興趣了解更多我的經歷，歡迎聯繫我並探討如何運用我的多樣技能為您的團隊成功作出貢獻。
