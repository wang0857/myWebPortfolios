As an enthusiastic junior web developer, I possess a solid foundation in HTML, CSS, and JavaScript and eager to leverage this knowledge in creating visually appealing and interactive websites. Furthermore, my professional background spans web publishing, UI development, and strategic marketing. Recently, I have been actively enhancing my front-end development skills by participating in Front-end Mentor Challenges. I utilized HTML, SCSS, React, Vue, and Vanilla JavaScript to create various web applications and interactive tools, including a tip calculator and interactive pricing component. These projects showcase my proficiency in developing user interfaces in alignment with provided mock-ups and specifications.

I thrive on collaborative projects and excel in dynamic environments. Let's connect and explore how my diverse skill set can contribute to your team's success.
